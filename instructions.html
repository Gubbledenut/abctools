ABC Transcription Tools Instructions

This ABC to notation converter was developed by Michael Eskin.

Full project source may be found on: GitHub.

Getting Started

New ABC - Provides a simple example tune in ABC to get you started

Open ABC File - Opens an .abc or .txt file from your computer and loads it into the editor

You may also paste some ABC into the text box or just start typing

On desktop systems, the text box can be resized by dragging the lower right corner

The ABC will be converted to notation in realtime and displayed below a set of control buttons:

Save ABC File - Saves the contents of the ABC editor to a file

The saved file will be in your browser's Downloads directory

Copy All - Copies all the ABC text to the system clipboard so you can easily paste it into social media posts, emails, or other tools

Hide Controls / Show Controls - Hides/Shows the control buttons

Notation Maximize/Minimize Arrows

If any text is present in the ABC text editing area, this double-arrow control shows up on the top right corner of the page

The control hides or shows the ABC text editing area and all controls so you can view your notation fullscreen

Click once to hide the ABC text editing area and all controls

Click again to show the ABC text editing area and all controls

Playing the Tunes

This tool doesn't have the ability to play the tunes

For now, I suggest you use the following site to play your ABC files:

https://editor.drawthedots.com/

It can also export MIDI and audio files of tunes

Using the "Copy All" command you can easily copy your ABC from this tool and paste it in the drawthedots tool to test playback

Once you are happy with the playback, copy the ABC from the drawthedots tool, clear the ABC in this tool using the "Clear" button, then paste the edited ABC back into this tool for further editing

Basic Controls

Transpose Down - Transposes the tune down one semitone

Transpose Up - Transposes the tune up one semitone

Show/Hide Advanced Controls - Shows/Hides a new panel with several additional ABC filtering and display controls (described below)

Show/Hide Sharing Controls - Shows/Hides a new panel (described below) with options for sharing your tune set via URL or QR code

Clear - Clears the ABC text area

Advanced Controls

ABC Filters:

These are enabled if Annotations, Text, or Chords are detected in the ABC

Show/Hide Annotations - Shows/Hides most ABC annotations like Discography, History, Tempo, etc.

Show/Hide Text - Shows/Hides any %%center or %%text based text in the ABC

Show/Hide Chords - Shows/Hides any chords entered in the ABC

Capo Settings: (only enabled if Mandolin, Bouzouki, Mandola, or Guitar tablature is being displayed)

Capo - Allows you to set a capo setting from 0 to 12 frets for the Mandolin, Bouzouki, Mandola, or Guitar tablature

Staff Display Settings:

Set Staff Spacing - Allows you to set a wider gap between the staffs

Notation/Tablature Display Controls

These control how the notation is displayed.

Sheet Music - Standard notation only

Note Names - Shows standard notation and note names below

Mandolin - Shows standard notation and tablature for Mandolin

Zouk GDAD - Shows standard notation and tablature for GDAD-tuned Bouzouki

Mandola - Shows standard notation and tablature for CGDA-tuned Mandola

Guitar - Shows standard notation and tablature for standard EADGBE-tuned guitar

DADGAD - Shows standard notation and tablature for DADGAD-tuned guitar

Whistle Tab - Shows standard notation and tablature for whistle in the key of D

PDF Export

Export PDF - Exports the currently displayed score as a high quality PDF file

PDF Format - Select whether you want One Tune / Page or Multiple Tunes / Page in the PDF and if you want page numbers included:

One Tune / Page - Each tune is placed on a new page in the PDF, includes page numbers

Multiple Tunes / Page - Tunes are added one after another down each page in the PDF, includes page numbers

One Tune - No Page # - Each tune is placed on a new page in the PDF, no page numbers

Multiple - No Page # - Tunes are added one after another down each page in the PDF, no page numbers

Note: Selecting multiple tunes per page may result in tunes staves crossing page boundaries

To manually force a tune to a new page in the PDF, add:

%%newpage

on a new line immediately at the end of tunes after which you want to force a page break

This will allow you to easily format a tune book where you want multiple tunes per page, but some tunes to not be split between pages

Sharing Controls

Share URL: Copying and pasting this link in a browser address bar will open the editor website and copy in the tune set ABC and tablature settings

For Mandolin, Bouzouki, Mandola, and Guitar tablature, the link also includes the capo setting

If the share URL link is too long to work in most browsers, the Share URL buttons will be disabled and you will see a message suggesting sharing fewer tunes

Test Share URL - Opens the Share URL in a new browser tab

Copy Share URL - Copies the Share URL to the system clipboard so you can easily paste it into social media posts, emails, or other tools

Save Share URL - Saves the Share URL to a text file

The saved file will be in your browser's Downloads directory

Generate QR Code - Once a share URL is generated, if it is short enough to fit in a QR code, this button is activated

Clicking the button will generate a QR code in .PNG format for the share URL and open it in a new browser tab where it can be saved or printed

Sharing Tunes Open Maximized

Shared tunes, when opened from a Share URL or QR code, are initially displayed with the ABC text editing area and controls hidden so you can see the music full-screen

Click the Maximize/Minimize arrow to show the ABC text editing area and controls

Reading the Generated QR Codes

All devices should easily be able to read the simpler QR codes generated for sets of one to three tunes

Most iOS devices are able to read even the most complex QR codes generated by this tool using the built-in Camera app

Some Android devices may have trouble reading the most complex codes with their built-in Camera app

I suggest trying the free Google Lens app, available on the Google Play Store:

Google Lens

iPhone/iPad Chord Entry Issue Fix

If you are trying to enter ABC chords, for example "D" or "Am", on an iPhone or iPad, iOS will, by default stick in "smart quotes" instead of standard double quotes

To fix this, go into your device's Settings and under the General/Keyboard tab, and turn off "Smart Punctuation"

After you do this, you will be able to add standard double quotes

Known Limitations

Note name tablature can handle key changes only at the beginning of a staff, not in the middle
