ABC Transcription Tools Instructions

This ABC to notation converter was developed by Michael Eskin.

Full project source may be found at: GitHub.

Getting Started

Open .ABC File - Opens an .abc or .txt file from your computer and loads it into the editor

You may also paste some ABC into the text box or just start typing

The ABC will be converted to notation in realtime and displayed below a set of control buttons:

Save .ABC File - Saves the contents of the ABC editor to a file

The saved file will be in your browser's Downloads directory

Playing the Tunes

This converter doesn't have the ability to play the tunes (yet).

For now, I suggest you use the following site to play your ABC files:

https://editor.drawthedots.com/

It can also export MIDI and audio files of the tune

You can copy your ABC from here and paste it there to test playback

Basic Controls

Transpose Down - Transposes the tune down one note letter name

Transpose Up - Transposes the tune up one note letter name

Advanced Controls - Clicking this shows a new panel with several additional ABC filtering and display controls (described below)

Sharing Controls - Clicking this shows a new panel (described below) with options for sharing your tune set via URL or QR code

Clear - Clears the ABC text area

Advanced Controls

ABC Filters:

Toggle Annotations - Shows/Hides most ABC annotations like Discography, History, etc.

Toggle Text - Shows/Hides any %%center or %%text based text in the ABC

Toggle Chords - Shows/Hides any chords entered in the ABC

Staff Display Settings:

Set Staff Spacing - Allows you to set a wider gap between the staffs

Notation/Tablature Display Controls

These control how the notation is displayed.

Sheet Music - Standard notation only

Note Names Tab - Shows standard notation and note names below

Mandolin Tab - Shows standard notation and tablature for Mandolin

Guitar EADGBE - Shows standard notation and tablature for standard EADGBE-tuned guitar

Guitar DADGAD - Shows standard notation and tablature for DADGAD-tuned guitar

Whistle Tab - Shows standard notation and tablature for whistle in the key of D

Display Size Options

These can be useful if you just want to screen capture the generated notation or review your work:

Mini

Small

Medium

Large - Perfect for printing

PDF Export

Export PDF - Exports the currently displayed score as a high quality PDF file

PDF Format - Select whether you want One Tune / Page or Multiple Tunes / Page in the PDF:

One Tune / Page - Each tune is placed on a new page in the PDF

Multiple Tunes / Page - Tunes are added one after another vertically down each page in the PDF

Note: Selecting multiple tunes per page may result in tunes staves crossing page boundaries

Sharing Controls

Share URL: Copying and pasting this link in a browser address bar will open the editor website and copy in the tune set ABC and tab settings

If the share URL link is too long to work in most browsers, you will see a message suggesting sharing fewer tunes

Test Share URL - Opens the Share URL in a new browser tab

Save Share URL - Saves the ShareURL to a file

The saved file will be in your browser's Downloads directory

Generate QR Code - Once a share URL is generated, if it is short enough to fit in a QR code, this button appears

Clicking the button will generate a QR code in .PNG format for the share URL and open it in a new browser tab where it can be saved or printed

Reading the Generated QR Codes

All devices should easily be able to read the simpler QR codes generated for sets of one to three tunes

Most iOS devices are able to read even the most complex QR codes generated by this tool using the built-in Camera app

Some Android devices may have trouble reading the most complex codes with their built-in Camera app

I suggest trying the free Google Lens app, available on the Google Play Store:

Google Lens

Forcing a Page Break After a Tune for Multiple Tunes / Page Format PDFs

To force a page break after a specific tune during PDF generation, add the following ABC at the bottom of the tune:

%%newpage

This will allow you to easily format a tune book where you want multiple tunes per page, but some tunes to not be split between pages

Printing

When printing the page using your browser's Print command, only the generated notation is printed

Setting the Display Size to Large and then printing from your browser can achieve some nice results

Known Limitations

Note name tablature can handle key changes only at the beginning of a staff, not in the middle

Text tab notes with natural indication show the wrong note name

In this case, abcjs is giving me the wrong note number to translate to text, so I am unable to fix at this time

Transpose up and down are unable to deal with text in chord annotations and produces odd results

Transpose up and down are done using just letter transposition with some intelligence around the octaves

The transposer doesn't have any real idea about the key, so may not produce the expected result for some cases, for example Bmin transposed up will result in Cmin, not C#min
